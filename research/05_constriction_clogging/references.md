# References — Project 5 (constriction clogging)

DOIs below should be re-verified against the Crossref API before any
manuscript submission (the same discipline applied to the stenosis paper).
arXiv entries are preprints.

## Clogging — simulation benchmark (the answer key)

- **Bielinski, C., Aouane, O., Harting, J. & Kaoui, B.** *Squeezing multiple
  soft particles into a constriction: transition to clogging.*
  arXiv:2110.13299 (2021).
  3-D LBM-IBM-FEM. State diagram in (D/d, Ca): rigid clog at D/d ≈ 3; soft
  escape for Ca > 0.005; `T_evac ∝ (d/D)^0.827, Ca^-0.108`.
